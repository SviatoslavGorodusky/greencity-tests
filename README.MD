# GreenCity Events Page Testing

### Опис завдання

Цей репозиторій створено в межах курсу QA/QC. Основна мета — розробка тест-кейсів для сторінки подій (Events) веб-застосунку GreenCity.

### Ресурси

* **Посилання на сайт:** [GreenCity](https://www.greencity.cx.ua/#/greenCity/events)
* **Документація:** [Swagger UI](https://www.greencity.cx.ua/swagger-ui.html#/)

### Тест-кейси

1.  **TestCase1:** Перевірка фільтрації подій за типом.
2.  **TestCase2:** Позитивна перевірка пошуку подій за назвою.
3.  **TestCase3:** Перевірка перемикання вигляду відображення подій (Grid/List).
4.  **TestCaseNeg:** Негативна перевірка пошуку (введення неіснуючої назви).
