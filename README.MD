# GreenCity UI Automation Project 

Проект з автоматизації тестування користувацького інтерфейсу (UI) платформи **GreenCity**. 
Ця робота демонструє використання сучасних підходів до тестування на Python з використанням Selenium та Pytest.

##  Мета проекту
Автоматизація сценаріїв фільтрації та пошуку подій на сторінці Events з дотриманням принципів чистого коду та професійної звітності.

## 🛠 Технологічний стек
- **Мова:** Python 3.12+
- **Фреймворк:** Pytest
- **Інструмент:** Selenium WebDriver
- **Архітектура:** Page Object Model (POM) + Component-based approach
- **Звітність:** Allure Report

##  Архітектура проекту
Проект організований за модульним принципом:
- `src/pages/` — Page Objects (логіка сторінок).
- `src/components/` — UI Components (Header, FilterPanel тощо). Усі компоненти наслідуються від `BaseComponent`.
- `tests/` — Тестові сценарії.
- `conftest.py` — Конфігурація драйвера та фікстури.
- `pytest.ini` — Налаштування фреймворку та інтеграція з Allure.
